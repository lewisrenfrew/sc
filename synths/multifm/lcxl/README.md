# multifm LCXL template

Top buttons must be configured as **toggle**. Bottom buttons must be **momentary**.

## CC mapping

| Control | CC | Function |
|---|---|---|
| Fader 1–8 | 5–12 | Volume (0–1.5) |
| Enc1 1–8 | 13–20 | Carrier frequency (30–600 Hz) |
| Enc2 1–8 | 21–28 | Modulator ratio (0.25–8.0×) |
| Enc3 1–8 | 29–36 | Modulation index (0.01–12.0) |
| Top button 1–8 | 37–44 | Decouple — toggle mode (LED on = frozen, LED off = ratio tracking) |
| Bottom button 1–8 | 45–52 | Recall snapshot (short press) / Save snapshot (hold 1.5s+) |
